🚀 CÓMO EMPEZAR EN 3 PASOS
═══════════════════════════════════════════════════════════════════════════════

PASO 1: EJECUTAR SCRIPT (2 minutos)
   1. Abre el archivo: schema.sql
   2. Copia TODO el contenido
   3. Ve a Supabase.com → Tu Proyecto → SQL Editor
   4. Pega el contenido
   5. Presiona "RUN" (o Ctrl+Enter)
   6. ¡Hecho! ✅

PASO 2: CONFIGURAR ENTORNO (3 minutos)
   1. Copia .env.example → .env
   2. Obtén credenciales de Supabase:
      - Settings → API → Project URL
      - Settings → API → anon public key
      - Settings → API → service_role key
   3. Llena los valores en .env
   4. npm install @supabase/supabase-js

PASO 3: USAR EN TU CÓDIGO (2 minutos)
   1. Abre ejemplos_queries.js
   2. Copia la función que necesites
   3. Úsala en tu proyecto
   4. ¡Funciona! 🎉
